== Description ==

This is very simple wordpress plugin to check whole information about your wordpress website. systeminfo plugin allow you to check your wordpress version, php configurations and many more features which are given below:-

***********************
**      FEATURES     **
***********************

	* Webserver Configuration
	* WordPress Configuration
	* WordPress Uploads/Constants
	* Session Configuration
	* PHP Configuration
	* PHP Extensions
	* Apache info
	* Site Info 
	* Plugins
	* Wordpress Active Theme info



=== INSTALLATIONS ===
***********************

   Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= What is the systeminfo Use for? =
Systeminfo use for quick view about your site information and your hosting server information. 
Also show your current theme information and Active Plugins list, Inactive Plugins List and Required updates etc.
